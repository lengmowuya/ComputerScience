# 纯函数
一类特别的函数，不改写传入的参数数据，不会产生网络请求，输入设备，输出设备，不调用其他非纯函数。只要是同样的输入，必定得到同样的输出。

# 高阶函数
一类特别的函数，参数或返回值是函数
